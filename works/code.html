standard_deviation

A standard deviation calculator I wrote entirely in Python.

Of all the many coding projects I've developed between mid 2020 and late 2021, I thought it would be nice to showcase this one based on how it was written.

I took a Probability and Statistics course in the Spring 2021 semester, and among the many things I learned in this class, one of them that stood out was standard deviation. When doing the process on paper, I personally believed it was a lot of work and values to keep track of. As a result, I've decided to write an algorithm that imitated the whole process of standard deviation based on my knowledge of the Python programming language at the time.

As a bonus, I added the option to find the median and range of the set of values inputted.
